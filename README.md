# Today's Movie Recommendations

Live app at [haqle314.github.io](https://haqle314.github.io/hom-nay-xem-gi)

## Instructions

### Run locally
```shell
npm install
npm run start
```

### Run tests
```shell
npm run test
```

### Run with docker
```shell
docker build -t hom-nay-xem-gi .
docker run -p 3000:3000 hom-nay-xem-gi
```
